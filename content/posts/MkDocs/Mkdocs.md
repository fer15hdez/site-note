# MkDocs
- MkDocs es un generador de sitios estáticos rápido y sencillo, escrito en Python, que sirve para crear documentación de proyectos utilizando Markdown para el contenido y un archivo de configuración YAML.  
## Comandos
- mkdocs: El paquete que hace la construccion del sitio a partir de los markdown.   
- mkdocs-material: El tema mas usado para mkdocs.  
`pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org mkdocs mkdocs-material`: Instala el paquete  
`mkdocs serve`: Levanta el servidor mkdocs
