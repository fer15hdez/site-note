## Connect to s3
- Create a user and a key password  
- ***AWS Configure*** Enter the key password.  
- Create a S3 bucket (is like boxes where is storing the tree directory and file). Once created we can create directory. 
- Create a acces point 

#### Command to connect 
***aws s3  <command> <source/path> <destiny/path (uri access point)>***
***aws s3 cp ./test.txt s3://arn:aws:s3:us-east-1:223965763113:accesspoint/s3-acces-point-web***

***To read a object by http browser is necessary to create bucket policy***
#### Policy example
{
	"Version": "2012-10-17",
	"Statement": [
		{
            "Sid": "Public-Policy-Web-s3",
			"Effect": "Allow",
			"Principal": {
				"AWS": "*"
			},
			"Action": [
				"s3:GetObject",
				"s3:GetObjectVersion"
			],
			"Resource": "arn:aws:s3:::bucket-test-web/*",
			"Condition": {
				"StringEquals": {
					"s3:ExistingObjectTag/type": "web"
				}
			}
		}
	]
}
