MkDocs

  • MkDocs es un generador de sitios estáticos rápido y sencillo, escrito en Python, que sirve para crear documentación de proyectos utilizando Markdown para el contenido y un archivo de configuración YAML.

Comandos

  • mkdocs: El paquete que hace la construccion del sitio a partir de los markdown.
  • mkdocs-material: El tema mas usado para mkdocs.
    pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org mkdocs mkdocs-material: Instala el paquete
    mkdocs serve: Levanta el servidor mkdocs

Cadenas - String

Funciones de Texto (Strings) en Python

Las cadenas de texto (strings) son secuencias inmutables de caracteres. Python ofrece una gran variedad de métodos y funciones para manipularlas. Aquí algunas de las más utilizadas:

Métodos Comunes de Cadenas

  • str.strip(): Elimina los espacios en blanco (espacios, tabulaciones, saltos de línea) del inicio y del final de la cadena.

    • " Hola Mundo ".strip() resulta en "Hola Mundo"

  • str.split(): Divide la cadena en una lista de subcadenas usando un delimitador (por defecto, cualquier espacio en blanco).

Make a virtual enviroment

python3 -m venv name_enviroment -> Make a virtual env in the present directory.
source name_enviroment/bin/activate -> Activate the enviroment.
python -m pip install name_of_package -> Install the packege.
pip list -> Mostrará todos los paquetes instalados en el entorno virtual
pip freeze > requirements.txt -> Export a list with all the package installed. The file “requirements.txt” is used to install the requirement.
python -m pip install -r requirements.txt -> To install the requirement from the “requirements.txt” file.
