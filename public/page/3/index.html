Manage imges

history: show the history of a image, how was build.

TEST

La Regla de Oro

  • Un test es código que ejecuta otro código y afirma (asegura) que el resultado es correcto.

Terminología Clave

  • Aserción (Assertion/Affirmation): El punto central del test. Es una llamada a una función (ej. assert.Equal(esperado, actual)) que verifica una condición. Si la condición es falsa, el test falla.

  • Fixture: Es el entorno o conjunto de datos preconfigurados que se necesitan para ejecutar un test (ej. un usuario, una conexión a base de datos, un archivo temporal).

VARIABLES

VARIABLES PRIMITIVAS

  • byte Stores whole numbers from -128 to 127
  • short Stores whole numbers from -32,768 to 32,767
  • int Stores whole numbers from -2,147,483,648 to 2,147,483,647
  • long Stores whole numbers from -9,223,372,036,854,775,808 to 9,223,372,036,854,775,807
  • float Stores fractional numbers. Sufficient for storing 6 to 7 decimal digits
  • double Stores fractional numbers. Sufficient for storing 15 to 16 decimal digits
  • boolean Stores true or false values
  • char Stores a single character/letter or ASCII values

STRING

  • indexOf(): method returns the index (the position) of the first occurrence of a specified text in a string (including whitespace).
  • The toUpperCase() method converts a string to upper case letters.
  • The toLowerCase() method converts a string to lower case letters.
  • charAt() method to access a character at a specific position in a string: txt.charAt(0)
  • trim(): elimina los espacios del inicio y del fin de la cadena de texto.

CONCATENACION STRING Y NUMEROS

String x = "10";
int y = 20;
String z = x + y;  // z will be 1020 (a String)
